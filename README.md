# GOOT - Get Off On Time
Windows app to auto shutdown PC in time.

<img width="486" height="323" alt="image" src="https://github.com/user-attachments/assets/eb62d0a4-08fa-482c-935a-35d700ef8aad" />


Feature:
- Auto shutdown PC on time count
- Auto shutdown PC when on time
- Auto shutdown PC when app open
- Password protect for schedule cancel

Next build:
- Remote control from Telegram Bot
- Automation Schedule
- Advance action more than Shutdown (Restart, Sleep, Signout)
