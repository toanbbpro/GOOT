# GOOT - Get Off On Time
The Winform app to auto shutdown PC in time.

<img width="506" height="473" alt="image" src="https://github.com/user-attachments/assets/15b543c1-d4ea-4d33-be2c-71bff64f9654" />

Feature:
- Auto shutdown PC on time count
- Auto shutdown PC when on time
- Auto shutdown PC when app open
- Password protect for schedule cancel
